description: Start the backend and frontend dev servers for Trixieverse
---

// turbo-all

1. Start the backend server:
```
npm run start
```
Run this in `d:\stuff\gittan\Trixieverse\simple-trixieverse`. Run as a background command (WaitMsBeforeAsync: 3000).

2. Start the frontend dev server:
```
npm run dev
```
Run this in `d:\stuff\gittan\Trixieverse\simple-trixieverse`. Run as a background command (WaitMsBeforeAsync: 5000).

3. Confirm both servers are running by checking command status. Report the URLs to the user.
