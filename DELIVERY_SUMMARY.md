## What This System Does

**Input:** Player says "Get me from Gold to Platinum in my main role"

**Process:**
1. Analyze 50M+ historical matches
2. Extract champion statistics by tier
3. Calculate power spikes and item builds
4. Train ML models on winning paths
5. Generate personalized blueprint

**Output:** "Master Ahri with 87% climb probability in ~40 hours"

```bash
# 1. Create database tables
npm run migrate

# 2. Start ML service
cd ml && python main.py

# 3. Start backend (new terminal)
npm run dev

# 4. Test it

  -H "Authorization: Bearer <token>" \
  -d '{"targetTier":"PLATINUM","role":"MID"}'
```
